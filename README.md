# Stripe example

## Stripe setup
- step an account on stripe https://stripe.com

## App Configuration
- Config Merchant Id, public key, private key for braintree.connect
- Open file server/app.js

## Get started
- Start server: `node server/app.js`
- Start client: `http-server client`
